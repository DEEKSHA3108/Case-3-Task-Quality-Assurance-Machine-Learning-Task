This function "dictionary_to_text" converts a dictionary-like string nto a text format where key-value pairs are joined together with an underscore (_) and separated by commas.

Input data: {'item1': 'Red', 'item2': 'Red', 'item3': 'Red', 'item4': 'Red', 'item5': 'Red', 'item6': 'uncategorized'}

Output data: 'item1_Red', 'item2_Red', 'item3_Red', 'item4_Red', 'item5_Red', 'item6_uncategorized'

The .map() function applies the dictionary_to_text function on each element on the dataset

# Function to Convert Dictionary-Like Strings to Text
def dictionary_to_text(data):
    if isinstance(data, str) and data.startswith("{"):
        try:
            parsed_data = ast.literal_eval(data)
            if isinstance(parsed_data, dict):
                return ', '.join([f"{key}_{value}" for key, value in parsed_data.items() if value is not None])
        except (ValueError, SyntaxError):
            pass
    return data

Building an Autoencoder for Feature Learning

Define the Autoencoder Structure

- input_dim: The number of features in the dataset.
- encoding_dim = 64: The bottleneck layer size, meaning the data will be compressed into 64-dimensional representations.
- Input(shape=(input_dim,)): Defines the input layer matching the feature size.
- GaussianNoise(0.1)(input_layer): Adds noise for robustness, helping the model generalize.
- Dense(256, activation='relu', kernel_regularizer='l2'): First dense layer with L2 regularization to prevent overfitting.
- BatchNormalization(): Normalizes activations, speeding up training.
- Dropout(0.6): Drops 60% of neurons randomly to prevent overfitting.
- Dense(128, activation='relu', kernel_regularizer='l2'): Second layer with the same setup.
- encoded = Dense(encoding_dim, activation='relu'): Final encoded (compressed) representation.

Define Decoder (Mirrors the encoder but in reverse)

- Final layer (sigmoid): Outputs values between 0 and 1 to reconstruct the original input.

Compile the Autoencoder

- Creates the autoencoder model using the input layer and the reconstructed output (decoded).
- Adam optimizer with learning rate 0.0003: Balances fast learning with stability.
- Loss function: Mean Squared Error (MSE) is used to minimize reconstruction error.

Add Early Stopping

- early_stopping = EarlyStopping(monitor="loss", patience=5, restore_best_weights=True)

Train the Autoencoder

- Trains the autoencoder on the unlabeled dataset (X_unlabeled_scaled).
- Epochs = 50: The maximum number of training cycles.
- Batch size = 32: Trains in mini-batches of 32 for efficiency.
- Autoencoder learns to reconstruct X_unlabeled_scaled, extracting useful feature representations

Save the Autoencoder

- autoencoder.save("autoencoder_model.h5")
